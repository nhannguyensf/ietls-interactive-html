# IELTS Practice App

Static IELTS listening and reading practice pages, ready for GitHub Pages hosting.

## Pages

- Listening: `app/listening_test.html`
- Reading: `app/reading_test.html`

The root `index.html` links to both tests.

## GitHub Pages

1. Push this repository to GitHub.
2. Go to **Settings > Pages**.
3. Select **Deploy from a branch**.
4. Choose your branch, usually `main`.
5. Choose `/ (root)` as the folder.
6. Save and open the published Pages URL.

## Local Sharing

Double-click `START_APP.bat`, then open:

```text
http://127.0.0.1:8080/app/listening_test.html
```

To rebuild the share zip after changes, double-click `PACKAGE_APP.bat`.
